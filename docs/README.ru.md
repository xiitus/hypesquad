## 📋 Предварительные требования

Для работы инструмента необходим **Node.js**. Если он ещё не установлен, установите его сначала.

### 🍎 Mac

1. Откройте **Терминал**
   - Нажмите `Cmd + Пробел`, введите «Терминал» и нажмите Enter
2. Вставьте следующую команду и нажмите Enter:

```bash
brew install node
```

> 💡 Если `brew` недоступен, скачайте установщик с [официального сайта Node.js](https://nodejs.org/).

### 🪟 Windows

1. Перейдите на [официальный сайт Node.js](https://nodejs.org/)
2. Нажмите **зелёную кнопку «LTS»** для загрузки
3. Откройте скачанный файл и следуйте инструкциям по установке
4. **Перезагрузите компьютер** после установки

### ✅ Проверка установки

Откройте Терминал (Windows: **PowerShell**) и выполните:

```bash
node -v
```

Если отображается что-то вроде `v20.xx.x`, всё готово 👍

## 🚀 Как использовать (3 шага)

### Шаг 1 — 🔑 Получите токен Discord

«Токен» — это длинная строка, которая идентифицирует вашу учётную запись Discord. Следуйте этим шагам, чтобы скопировать его.

1. Откройте [Discord](https://discord.com/channels/@me) в **браузере** (Chrome, Brave, Edge и т.д.)
   - ⚠️ Используйте **браузерную версию**, а не настольное приложение

2. Откройте **инструменты разработчика**
   - Windows: нажмите `F12`
   - Mac: нажмите `Cmd + Option + I` (все три клавиши одновременно)

3. Справа (или внизу) появится панель → нажмите на вкладку **«Network»** вверху
   - Если не видите, нажмите `>>` чтобы показать скрытые вкладки

4. Вернитесь в Discord и **отправьте любое сообщение** в любом канале
   - Подойдёт что угодно, например «привет» или «test»

5. В инструментах разработчика слева появится запрос **`messages`** → **нажмите на него**

6. Справа появятся подробности → нажмите на вкладку **«Headers»**

7. Прокрутите вниз и найдите раздел **«Request Headers»**

8. Найдите **`Authorization`** и **правый клик → копировать** длинную строку рядом ✅

> ⚠️ **Никогда не передавайте свой токен другим людям!**
> С вашим токеном кто угодно может полностью управлять вашим аккаунтом.
> Обращайтесь с ним как с паролем 🔐

---

### Шаг 2 — ⚡ Выполните команду

Откройте Терминал (Windows: PowerShell), вставьте следующее и нажмите Enter:

```bash
npx hypesquad --bravery
```

> 💡 Замените `--bravery` на `--brilliance` или `--balance` для другого House.

Появится запрос:

```
Discord token:
```

**Вставьте** токен, скопированный на шаге 1, и нажмите Enter.

- Windows: `Ctrl + V` для вставки
- Mac: `Cmd + V` для вставки

🔒 При вводе отображается `****` (из соображений безопасности символы не видны).

Если вы видите `Done: Set to bravery (204)`, всё получилось 🎉

---

### Шаг 3 — ✅ Проверьте значок

1. Нажмите на свой аватар в Discord → откройте **Профиль**
2. Если значок HypeSquad отображается — готово! 🥳

## 🧪 Пробный запуск (режим предпросмотра)

Посмотрите, что будет отправлено, без фактического изменения значка. Попробуйте сначала это, если не уверены:

```bash
npx hypesquad --bravery --dry-run
```

Результат:

```
[DRY RUN] POST https://discord.com/api/v9/hypesquad/online
  Action: bravery
  Token: abcd****************************
  Body: {"house_id":1}
```

Если всё в порядке, уберите `--dry-run` и выполните по-настоящему 👍

## ❓ Часто задаваемые вопросы

### «npx не найден»

→ Node.js не установлен. См. [📋 Предварительные требования](#-предварительные-требования).

### «Failed (401)»

→ Токен неверный или просрочен. Повторите шаг 1.

### «Failed (429)»

→ Слишком много запросов за короткое время. Подождите несколько минут и попробуйте снова.

### Значок не изменился

→ Закройте и снова откройте Discord — обновление может занять некоторое время.
